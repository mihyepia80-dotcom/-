# 2026-1학기 학교 중간평가 웹앱

부서별·학년별 중간평가 및 협의록 작성 웹 애플리케이션입니다.

## 서식 구성

| 번호 | 서식명 | 설명 |
|------|--------|------|
| 1~3 | 학년별 협의록 | 1·2·3학년 교육활동 평가 + 첨부 양식 |
| 4 | 교과/비교과 평가 | 본인·주관·행사·업무 평가 + 첨부 8항목 |
| 5 | 행사 등 확인 | 월별 행사 추진결과 및 보완점 |
| 6 | 부서별 협의록 | 8개 부서 담당자·업무·추진결과 (사전 입력) |

## 실행 방법

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.

```bash
python -m http.server 8080
# http://localhost:8080
```

## 기능

- 6개 서식 탭 전환
- 입력 내용 자동 저장 (localStorage)
- 행 추가·삭제
- 인쇄 지원 (A4 가로)
- 6번 서식: 8개 부서 담당자·업무내용 사전 입력

## 파일 구조

```
index.html
css/styles.css
js/
  utils.js        — 공통 유틸
  form-grade.js   — 1~3학년 협의록
  form4.js        — 교과/비교과
  form5.js        — 행사 확인
  form6-data.js   — 부서별 담당자 데이터
  form6.js        — 부서별 협의록
  app.js          — 탭·초기화
```
