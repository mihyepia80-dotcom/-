# 2026-1학기 학교 중간평가 웹앱

부서별 월별 중간평가, 담당업무 중간평가, 교육활동 평가 협의록을 작성하는 웹 애플리케이션입니다.

## 서식 구성

1. **월별 중간평가** — 부서별(담당자) 월별 중간평가 항목 작성
2. **담당업무 중간평가** — 부서별 담당업무 담당자 중간평가 항목 작성
3. **교육활동 협의록** — 2026학년도 1학기 교육활동 평가 협의록(학년)
4. **학년별 첨부** — 1~3번 서식용 학년별 협의록 첨부 양식 (1·2·3학년)
5. **교과/비교과 평가** — 2026학년도 1학기 교육활동 평가(교과/비교과용)
6. **행사 확인** — 2026-1학기 행사 등 확인
7. **부서별 협의록** — 8개 부서(교무기획부~6학년부) 담당자·업무내용 사전 입력

## 실행 방법

별도 설치 없이 브라우저에서 바로 실행할 수 있습니다.

1. `index.html` 파일을 더블클릭하거나
2. 브라우저 주소창에 파일 경로를 입력하여 열기

또는 로컬 서버로 실행:

```bash
# Python이 설치된 경우
python -m http.server 8080
# http://localhost:8080 접속
```

## 기능

- 3가지 평가 서식 탭 전환
- 입력 내용 자동 저장 (브라우저 localStorage)
- 행/업무 추가·삭제
- 인쇄 지원 (A4 가로)
